TASK:

Design an AI agent that helps a user plan learning in a structured and adaptive manner. 
An example goal could be learning Data Science within 30 days. Functional Requirements :-
The agent must decompose the learning goal into smaller learning units and milestones.
The agent must retrieve reference learning materials and topic structures from curated knowledge sources.
The agent must estimate required effort and build a day-wise study schedule using user constraints and preferences.
The agent must accept user feedback and revise the remaining learning plan accordingly. 
The agent must preserve user history and planning context for future interactions. Expected Behaviour of the Agent :-
The agent must decide how learning units should be structured and sequenced. 
The agent must evaluate whether the generated schedule is feasible. 
The agent must decide which parts of the learning plan need revision after feedback.

Your Task :- Design a complete AI agent architecture. Draw a flow showing planning, evaluation, feedback handling,
and plan revision. Clearly show multiple internal components involved in the process. 
Mention the libraries and frameworks that would be used and justify the choices.
